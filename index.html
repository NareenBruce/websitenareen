FocusMirror — Real-Time Focus Tracker

A real-time focus-tracking app that uses your webcam to classify your state — Focused, Distracted (phone in hand), Sleepy, or Away — and logs it across a work session. Fuses MediaPipe Face Mesh (Eye/Mouth Aspect Ratio for drowsiness) with GPU-accelerated YOLOv8 phone detection through a custom state machine with debounced smoothing. A FastAPI WebSocket backend streams live results to a Next.js/TypeScript frontend. Session summary exports to CSV with time-per-state, phone pickups, and longest focus streak. Fully local — no video ever leaves the machine.

Smart Road Hazard Detection System

An automated real-time computer vision system that detects road hazards — potholes, cracks, and open manholes — from live dashcam feeds. Trained a lightweight YOLOv8n model on ~2,700 annotated images with transfer learning from COCO and data augmentation. Achieved 75.9% mAP@0.5 at ~30 FPS real-time inference. Integrated into a full-stack web app with a Python FastAPI backend and Next.js frontend, featuring a live detection dashboard, detection history log, and bounding box overlays with confidence scores.

Bank Lead AI — Marketing Prediction Model

A data mining solution to predict customer subscription success using the UCI Bank Marketing dataset. Built and compared Logistic Regression, Decision Tree, Random Forest, and Gradient Boosting models. Addressed severe class imbalance using SMOTE. Best performance: Random Forest with ROC-AUC of 0.783. Deployed as a Streamlit application supporting real-time single-customer prediction and bulk CSV analysis for marketing teams.
